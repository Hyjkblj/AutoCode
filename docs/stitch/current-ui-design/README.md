# Current UI Design Export

This folder contains the current Stitch UI design export for the current mobile full-stack automation project.

- `index.html`: local gallery for all exported screens
- `manifest.json`: exported screen manifest with route, HTML, and PNG paths
- `project.json`: Stitch project metadata and design theme
- `routes.json`: screen-to-route mapping used for the export
- `screens/`: raw HTML for each exported screen
- `images/`: screenshot PNG for each exported screen

Export source: Stitch MCP via `@google/stitch-sdk`
