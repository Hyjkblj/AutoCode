目标
- 替换 HTTP 轮询，改为 STOMP/WebSocket 订阅实时事件流。
- 断线后自动重连，通过 lastSeq 补回断线期间的事件。

技术方案（使用 krossbow，不自己写 STOMP 帧解析）

  // 连接示例（krossbow + OkHttp）
  val client = StompClient(OkHttpWebSocketClient())
  val session = client.connect("ws://$baseUrl/ws",
      customStompConnectHeaders = mapOf("Authorization" to "Bearer $token"))
  val subscription = session.subscribe("/topic/tasks/$taskId",
      converter = SimpleTextMessageConverter)
  subscription.collect { frame ->
      val event = Json.decodeFromString<EventDto>(frame.body ?: return@collect)
      onEvent(event)
  }

- 断线重连：krossbow 内置重连策略，配置 reconnectDelay 即可。
- 补流：断线重连后调用 GET /api/v1/tasks/{taskId}/events?lastSeq={n} 补历史。

新增/变更内容
- build.gradle.kts：新增 krossbow-stomp-core + krossbow-websocket-okhttp
- network/StompEventClient.kt（新增，封装 krossbow）
  - connect(taskId, token, lastSeq, onEvent)
  - disconnect()
- network/EventDto.kt（新增）
- AppViewModel.kt：新增 subscribeTaskEvents / unsubscribeTaskEvents
- AppUi.kt：TaskDetailTab 进入时订阅，离开时取消

事件展示规则
- TASK_STARTED       → 状态提示行
- ASSISTANT_OUTPUT   → 气泡卡片（payload.message + agent 名称）
- TOOL_START         → 工具调用卡片（可折叠，展示 tool/command/cwd）
- TOOL_END           → 执行结果行（exitCode/耗时/颜色）
- FILE_PATCH_PREVIEW → Markdown 代码块（MOB-05 渲染）
- APPROVAL_REQUIRED  → 触发 MOB-02 审批弹窗
- TASK_DONE/FAILED   → 终态横幅

- 断线重连后 lastSeq 正确，无重复事件
- 离线模式降级为本地 ticker，不报错

技术方案（accompanist-permissions 处理权限，系统 SpeechRecognizer 识别）

  // 权限申请（accompanist，不自己写 ActivityResultContracts 样板代码）
  val audioPermState = rememberPermissionState(Manifest.permission.RECORD_AUDIO)

  // 语音识别（系统级，无第三方 SDK）
  val recognizerIntent = Intent(RecognizerIntent.ACTION_RECOGNIZE_SPEECH).apply {
      putExtra(RecognizerIntent.EXTRA_LANGUAGE, "zh-CN")
      putExtra(RecognizerIntent.EXTRA_LANGUAGE_MODEL, RecognizerIntent.LANGUAGE_MODEL_FREE_FORM)
  }
  val launcher = rememberLauncherForActivityResult(
      ActivityResultContracts.StartActivityForResult()
  ) { result ->
      val text = result.data?.getStringArrayListExtra(RecognizerIntent.EXTRA_RESULTS)?.firstOrNull()
      if (text != null) onVoiceResult(text)
  }

  Card(colors = CardDefaults.cardColors(containerColor = agentColor(event.agentName))) {
      Row {
          AsyncImage(model = agentIconUrl(event.agentName), contentDescription = null)  // coil3
          Text(event.payload["message"] ?: "")
      }
  }

  // TOOL_START 可折叠卡片：用 AnimatedVisibility，不引入额外库
  var expanded by remember { mutableStateOf(false) }

          Text("▶ ${event.payload["tool"]}  ${event.payload["command"]}")
          AnimatedVisibility(expanded) {
